GIT

VSC - Version Control System.

VSC бывают централизованные  (centralized)  и распределенные  (distributed).
Centralized - когда полная исторя проэкта хранится только на сервере.
Distributed - когда у каждого разработчика на лок. компьютере хранится проект с полной историей изменений в проекте.

GIT - относится к Distributed VSC, это набор коммитов, связанных между собой ссылками. Каждый commit связан ссылкой с предыдущим commit и так формируется история.

Каждый раз при сохранении проекта в GIT, он запоминает как выглядит каждый файл, делая его снимок и вычисляет его хеш сумму и сохраняет ссылку на этот снимок.
